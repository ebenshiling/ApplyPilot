ApplyPilot Quick Start

Simple guide for non-technical users: setup, run, and get tailored resumes fast.

1) Pick Your Setup

Option A (Recommended): Docker

docker compose up --build

Open: http://127.0.0.1:8765

Option B: Python install

python scripts/bootstrap.py
applypilot dashboard-serve --multi-user
If you share this system with multiple people, always run dashboard-serve --multi-user.

2) Sign In and Complete Setup

  1. Open dashboard and login/register.
  2. Go to Setup.
  3. In Full Profile Editor, fill your real details.
  4. Save your profile.
  5. Paste your resume into resume.txt field and save.
  6. Set job search options in Search Config builder or advanced YAML.

3) Add API Key (Required for Tailoring)

Add your key in your workspace .env:

GEMINI_API_KEY=your_real_key_here

Docker multi-user path pattern:

docker-data/multi/workspaces/<your-username>/.env

4) Run the Pipeline

Use Pipeline Controls in dashboard:

  1. Run Discover
  2. Run Enrich
  3. Run Score
  4. Review jobs with score 7+
  5. Pick the jobs you want
  6. Enable Selected only
  7. Run Tailor (or use Regenerate tailored resumes now)

5) Daily Use Pattern

StepWhat to do
MorningDiscover + Enrich + Score
MiddayPick best jobs and tailor selected only
AfternoonApply (manual or auto-apply mode)
EveningBackup with workspace-export

6) Most Useful Commands

applypilot status
applypilot jobs --limit 50
applypilot run discover enrich score
applypilot run tailor cover pdf
applypilot apply --dry-run
applypilot workspace-export --out ~/applypilot-transfer.zip --include-env --include-generated

7) Common Issues (Fast Fix)

  • "requires Tier 2": missing API key in user workspace .env.
  • "csrf_failed": refresh/login again, then retry; ensure latest dashboard build.
  • No jobs returned: widen locations, roles, and results per site.
  • Tailoring too generic: improve Full Profile + Resume Facts with real metrics/projects.

8) Safety Rules

  • Never commit or share real API keys publicly.
  • Use separate user accounts in multi-user mode.
  • Keep profile facts truthful; do not fabricate experience.
